📋 Design Principles Summary
🫁 Breathable Whitespace
Padding generosa (20-28px), margin antar elemen minimal 12px, tidak ada elemen yang "nempel". Ruang kosong adalah bagian dari desain, bukan kekurangan.
🎯 Youth-Focused
Font display modern (Space Grotesk), warna bold tapi terkontrol, animasi halus (pulse, breathe), dan language yang conversational.
🛡️ Safety-First Visuals
SOS selalu merah (#E11D48) dengan animasi pulse. Guardian selalu ungu (#7C3AED). Tidak ada warna yang ambigu untuk fitur keamanan.
📱 Consistent Patterns
Bottom nav 4 tab tetap, header dengan actions kanan, rounded corners konsisten (12-16px), dan dark overlays untuk mode kritis.
♿ Accessibility
Touch target min 40x40px, kontras memenuhi WCAG 4.5:1, font scalable, dan SOS bisa diakses bahkan saat terkunci.
🌙 Dark Mode Ready
Struktur warna modular memudahkan dark mode: swap --c-bg ↔ --c-text, --c-surface ↔ --c-surface-3. Video screen sudah dark by default.
